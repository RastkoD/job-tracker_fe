# JobTrckr 💼

A full-stack job application tracking web app built to manage, organize, and analyze job search activity in a single dashboard.  
The project focuses on clean UX, efficient CRUD workflows, authentication, and real-world application logic.

---

## 🧠 Project Focus

- Job application tracking and organization
- Authentication-protected user workflows
- Real-time CRUD interactions
- Status-based analytics and insights
- Clean, minimal, productivity-focused UI

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- REST API integration
- Supabase-backed backend service

---

## ✨ Key Features

- Admin-authenticated CRUD access for managing job entries
- Create, edit, update, and delete job entries
- Application dashboard with filtering and status tracking
- Summary statistics and progress insights
- Public demo mode with read-only access to applications and stats
- Responsive, distraction-free interface
- API-driven real-time data updates

---

## 🧩 Implementation Highlights

- Component-driven React architecture
- Role-based behavior (admin vs public viewer)
- Centralized state and form handling
- REST API consumption with async workflows
- Optimistic UI updates for smooth interaction
- Modular UI components for scalability
- Error handling and loading state management
